ΠΡΟΕΠΙΣΚΟΠΗΣΗ ΛΥΣΗΣ

Λύση: ΘΕΜΑ Γ (Ηλεκτρονικό Κατάστημα - Πανελλήνιες 2022)

Πρόγραμμα και Συνάρτηση:

ΠΡΟΓΡΑΜΜΑ πανελλήνιες2022Γ
ΜΕΤΑΒΛΗΤΕΣ
  ΑΚΕΡΑΙΕΣ: απ1, απ2, πλΔεν, πλΜαθ, αρπρο
  ΠΡΑΓΜΑΤΙΚΕΣ: τ1, τ2, ποσοστό, έσοδα
ΑΡΧΗ
  ΓΡΑΨΕ 'Δώσε απόθεμα πρ 1'
  ΔΙΑΒΑΣΕ απ1
  ΟΣΟ απ1 <= 0 ΕΠΑΝΑΛΑΒΕ
    ΓΡΑΨΕ 'Δώστε θετικό αριθμό'
    ΔΙΑΒΑΣΕ απ1
  ΤΕΛΟΣ_ΕΠΑΝΑΛΗΨΗΣ
  ΓΡΑΨΕ 'Δώσε απόθεμα πρ 2'
  ΔΙΑΒΑΣΕ απ2
  ΟΣΟ απ2 <= 0 ΕΠΑΝΑΛΑΒΕ
    ΓΡΑΨΕ 'Δώστε θετικό αριθμό'
    ΔΙΑΒΑΣΕ απ2
  ΤΕΛΟΣ_ΕΠΑΝΑΛΗΨΗΣ
  ΓΡΑΨΕ 'Δώστε τιμή προϊόντος 1 και 2'
  ΔΙΑΒΑΣΕ τ1, τ2
  έσοδα <- 0
  ποσοστό <- 0
  πλΜαθ <- 0
  πλΔεν <- 0
  ΟΣΟ (απ1 > 0 Η απ2 > 0) ΚΑΙ ποσοστό <= 20 ΕΠΑΝΑΛΑΒΕ
    πλΜαθ <- πλΜαθ + 1
    ΓΡΑΨΕ 'Δώσε αριθμό προϊόντος'
    ΔΙΑΒΑΣΕ αρπρο
    ΑΝ ΥΠΑΡΧΕΙ(αρπρο, απ1, απ2) = ΑΛΗΘΗΣ ΤΟΤΕ
      ΑΝ αρπρο = 1 ΤΟΤΕ
        απ1 <- απ1 - 1
        έσοδα <- έσοδα + τ1
        ΓΡΑΨΕ 'Εξυπηρετήθηκε πήραμε ', τ1, ' ευρώ'
      ΑΛΛΙΩΣ
        απ2 <- απ2 - 1
        έσοδα <- έσοδα + τ2
        ΓΡΑΨΕ 'Εξυπηρετήθηκε πήραμε ', τ2, ' ευρώ'
      ΤΕΛΟΣ_ΑΝ
    ΑΛΛΙΩΣ
      ΓΡΑΨΕ 'Δεν εξυπηρετήθηκε'
      πλΔεν <- πλΔεν + 1
    ΤΕΛΟΣ_ΑΝ
    ποσοστό <- πλΔεν * 100 / πλΜαθ
  ΤΕΛΟΣ_ΕΠΑΝΑΛΗΨΗΣ
  ΓΡΑΨΕ 'Τα συνολικά έσοδα του καταστήματος ήταν ', έσοδα
ΤΕΛΟΣ_ΠΡΟΓΡΑΜΜΑΤΟΣ

ΣΥΝΑΡΤΗΣΗ ΥΠΑΡΧΕΙ(αρπρο, απ1, απ2): ΛΟΓΙΚΗ
ΜΕΤΑΒΛΗΤΕΣ
  ΑΚΕΡΑΙΕΣ: αρπρο, απ1, απ2
ΑΡΧΗ
  ΥΠΑΡΧΕΙ <- ΨΕΥΔΗΣ
  ΑΝ αρπρο = 1 ΚΑΙ απ1 > 0 ΤΟΤΕ
    ΥΠΑΡΧΕΙ <- ΑΛΗΘΗΣ
  ΑΛΛΙΩΣ_ΑΝ αρπρο = 2 ΚΑΙ απ2 > 0 ΤΟΤΕ
    ΥΠΑΡΧΕΙ <- ΑΛΗΘΗΣ
  ΤΕΛΟΣ_ΑΝ
