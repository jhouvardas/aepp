ΠΡΟΕΠΙΣΚΟΠΗΣΗ ΛΥΣΗΣ

Λύση Μεζεδακίου: ΘΕΜΑ 4ο (Παραγωγή Αυγών)

Κύριο Πρόγραμμα και Συνάρτηση:

ΠΡΟΓΡΑΜΜΑ Παραγωγη_Αυγων
ΜΕΤΑΒΛΗΤΕΣ
  ΑΚΕΡΑΙΕΣ: ετοςΠαρακολουθησης, ημερεςΑναΜηνα[12], i, j, παραγωγηΑυγων[12, 31], συνολοΜηνα, συνολοΕτους
ΑΡΧΗ
  ΑΡΧΗ_ΕΠΑΝΑΛΗΨΗΣ
    ΓΡΑΨΕ 'Δώστε το έτος παρακολούθησης (2000-2099):'
    ΔΙΑΒΑΣΕ ετοςΠαρακολουθησης
  ΜΕΧΡΙΣ_ΟΤΟΥ ετοςΠαρακολουθησης >= 2000 ΚΑΙ ετοςΠαρακολουθησης <= 2099

  ΓΙΑ i ΑΠΟ 1 ΜΕΧΡΙ 12
    ημερεςΑναΜηνα[i] <- ΗΜΕΡΕΣ_ΜΗΝΑ(ετοςΠαρακολουθησης, i)
  ΤΕΛΟΣ_ΕΠΑΝΑΛΗΨΗΣ

  ΓΙΑ i ΑΠΟ 1 ΜΕΧΡΙ 12
    ΓΡΑΨΕ 'Καταχώριση παραγωγής για τον μήνα: ', i
    ΓΙΑ j ΑΠΟ 1 ΜΕΧΡΙ ημερεςΑναΜηνα[i]
      ΓΡΑΨΕ 'Δώστε την παραγωγή για την ημέρα ', j, ':'
      ΔΙΑΒΑΣΕ παραγωγηΑυγων[i, j]
    ΤΕΛΟΣ_ΕΠΑΝΑΛΗΨΗΣ
  ΤΕΛΟΣ_ΕΠΑΝΑΛΗΨΗΣ

  συνολοΕτους <- 0
  ΓΙΑ i ΑΠΟ 1 ΜΕΧΡΙ 12
    συνολοΜηνα <- 0
    ΓΙΑ j ΑΠΟ 1 ΜΕΧΡΙ ημερεςΑναΜηνα[i]
      συνολοΜηνα <- συνολοΜηνα + παραγωγηΑυγων[i, j]
    ΤΕΛΟΣ_ΕΠΑΝΑΛΗΨΗΣ
    ΓΡΑΨΕ 'Παραγωγή μήνα ', i, ': ', συνολοΜηνα
    συνολοΕτους <- συνολοΕτους + συνολοΜηνα
  ΤΕΛΟΣ_ΕΠΑΝΑΛΗΨΗΣ

  ΓΡΑΨΕ 'Συνολική ετήσια παραγωγή: ', συνολοΕτους

ΤΕΛΟΣ_ΠΡΟΓΡΑΜΜΑΤΟΣ

ΣΥΝΑΡΤΗΣΗ ΗΜΕΡΕΣ_ΜΗΝΑ(ετος, μηνας) : ΑΚΕΡΑΙΑ
ΜΕΤΑΒΛΗΤΕΣ
  ΛΟΓΙΚΕΣ: δισεκτο
ΑΡΧΗ
  δισεκτο <- (ετος MOD 4 = 0 ΚΑΙ ετος MOD 100 <> 0) Η (ετος MOD 400 = 0)

  ΑΝ μηνας = 2 ΤΟΤΕ
    ΑΝ δισεκτο = ΑΛΗΘΗΣ ΤΟΤΕ
      ΗΜΕΡΕΣ_ΜΗΝΑ <- 29
    ΑΛΛΙΩΣ
      ΗΜΕΡΕΣ_ΜΗΝΑ <- 28
    ΤΕΛΟΣ_ΑΝ
  ΑΛΛΙΩΣ_ΑΝ μηνας <= 7 ΤΟΤΕ
    ΑΝ μηνας MOD 2 <> 0 ΤΟΤΕ
      ΗΜΕΡΕΣ_ΜΗΝΑ <- 31
    ΑΛΛΙΩΣ
      ΗΜΕΡΕΣ_ΜΗΝΑ <- 30
    ΤΕΛΟΣ_ΑΝ
  ΑΛΛΙΩΣ
    ΑΝ μηνας MOD 2 <> 0 ΤΟΤΕ
      ΗΜΕΡΕΣ_ΜΗΝΑ <- 30
    ΑΛΛΙΩΣ
      ΗΜΕΡΕΣ_ΜΗΝΑ <- 31
    ΤΕΛΟΣ_ΑΝ
  ΤΕΛΟΣ_ΑΝ
