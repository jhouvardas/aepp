Λύση Θέματος 4 - Διαχείριση Παραγωγής Αυγών

ΠΡΟΓΡΑΜΜΑ παραγωγήΑυγών
ΜΕΤΑΒΛΗΤΕΣ
  ΑΚΕΡΑΙΕΣ: έτος, μήνας, μέρα, πλΗμερών[12], παραγωγή[12, 31], σύνΜήνα, σύνΈτους
ΑΡΧΗ
  ! Α1: Είσοδος έτους με έλεγχο εγκυρότητας (άγνωστες επαναλήψεις)
  έτος <- 0
  ΟΣΟ έτος < 2000 Η έτος > 2099 ΕΠΑΝΑΛΑΒΕ
    ΓΡΑΨΕ 'Δώστε έτος (2000-2099):'
    ΔΙΑΒΑΣΕ έτος
  ΤΕΛΟΣ_ΕΠΑΝΑΛΗΨΗΣ

  ! Υπολογισμός ημερών κάθε μήνα (γνωστές επαναλήψεις)
  ΓΙΑ μήνας ΑΠΟ 1 ΜΕΧΡΙ 12
    πλΗμερών[μήνας] <- ημέρεςΜήνα(έτος, μήνας)
  ΤΕΛΟΣ_ΕΠΑΝΑΛΗΨΗΣ

  ! Α2: Είσοδος ημερήσιας παραγωγής
  ΓΙΑ μήνας ΑΠΟ 1 ΜΕΧΡΙ 12
    ΓΡΑΨΕ 'Μήνας: ', μήνας
    ΓΙΑ μέρα ΑΠΟ 1 ΜΕΧΡΙ πλΗμερών[μήνας]
      ΓΡΑΨΕ 'Μέρα ', μέρα, ', αυγά: '
      ΔΙΑΒΑΣΕ παραγωγή[μήνας, μέρα]
    ΤΕΛΟΣ_ΕΠΑΝΑΛΗΨΗΣ
  ΤΕΛΟΣ_ΕΠΑΝΑΛΗΨΗΣ

  ! Α3: Στατιστικά παραγωγής
  σύνΈτους <- 0
  ΓΙΑ μήνας ΑΠΟ 1 ΜΕΧΡΙ 12
    σύνΜήνα <- 0
    ΓΙΑ μέρα ΑΠΟ 1 ΜΕΧΡΙ πλΗμερών[μήνας]
      σύνΜήνα <- σύνΜήνα + παραγωγή[μήνας, μέρα]
    ΤΕΛΟΣ_ΕΠΑΝΑΛΗΨΗΣ
    ΓΡΑΨΕ 'Μήνας ', μήνας, ': ', σύνΜήνα
    σύνΈτους <- σύνΈτους + σύνΜήνα
  ΤΕΛΟΣ_ΕΠΑΝΑΛΗΨΗΣ
  ΓΡΑΨΕ 'Σύνολο έτους: ', σύνΈτους

ΤΕΛΟΣ_ΠΡΟΓΡΑΜΜΑΤΟΣ

! Β: Υποπρόγραμμα Συνάρτηση (camelCase)
ΣΥΝΑΡΤΗΣΗ ημέρεςΜήνα(έτος, μήνας) : ΑΚΕΡΑΙΑ
ΜΕΤΑΒΛΗΤΕΣ
  ΑΚΕΡΑΙΕΣ: έτος, μήνας
  ΛΟΓΙΚΕΣ: δίσεκτο
ΑΡΧΗ
  ΑΝ μήνας = 2 ΤΟΤΕ
    δίσεκτο <- (έτος MOD 4 = 0 ΚΑΙ έτος MOD 100 <> 0) Η (έτος MOD 400 = 0)
    ΑΝ δίσεκτο = ΑΛΗΘΗΣ ΤΟΤΕ
      ημέρεςΜήνα <- 29
    ΑΛΛΙΩΣ
      ημέρεςΜήνα <- 28
    ΤΕΛΟΣ_ΑΝ
  ΑΛΛΙΩΣ_ΑΝ μήνας <= 7 ΤΟΤΕ
    ΑΝ μήνας MOD 2 <> 0 ΤΟΤΕ
      ημέρεςΜήνα <- 31
    ΑΛΛΙΩΣ
      ημέρεςΜήνα <- 30
    ΤΕΛΟΣ_ΑΝ
  ΑΛΛΙΩΣ
    ΑΝ μήνας MOD 2 = 0 ΤΟΤΕ
      ημέρεςΜήνα <- 31
    ΑΛΛΙΩΣ
      ημέρεςΜήνα <- 30
    ΤΕΛΟΣ_ΑΝ
  ΤΕΛΟΣ_ΑΝ
