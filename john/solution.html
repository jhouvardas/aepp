ΠΡΟΕΠΙΣΚΟΠΗΣΗ ΛΥΣΗΣ

Λύση σε ΓΛΩΣΣΑ

ΠΡΟΓΡΑΜΜΑ Καφετέρια
ΜΕΤΑΒΛΗΤΕΣ
  ΑΚΕΡΑΙΕΣ: π[16], i, j, αρΤραπ
  ΠΡΑΓΜΑΤΙΚΕΣ: λογ[16, 5], συνΛογ
  ΛΟΓΙΚΕΣ: βρΑ, βρΓ, έχειΜηδέν
ΑΡΧΗ

  ΓΙΑ i ΑΠΟ 1 ΜΕΧΡΙ 16
    ΓΡΑΨΕ 'Δώσε πλήθος ατόμων για το τραπέζι ', i, ':'
    ΔΙΑΒΑΣΕ π[i]
    ΟΣΟ π[i] < 0 Η π[i] > 5 ΕΠΑΝΑΛΑΒΕ
      ΓΡΑΨΕ 'Λάθος τιμή. Δώσε αριθμό ατόμων από 0 έως 5:'
      ΔΙΑΒΑΣΕ π[i]
    ΤΕΛΟΣ_ΕΠΑΝΑΛΗΨΗΣ
    ΓΙΑ j ΑΠΟ 1 ΜΕΧΡΙ π[i]
      ΓΡΑΨΕ 'Λογαριασμός ατόμου ', j, ' στο τραπέζι ', i, ':'
      ΔΙΑΒΑΣΕ λογ[i, j]
      ΟΣΟ λογ[i, j] < 0 ΕΠΑΝΑΛΑΒΕ
        ΓΡΑΨΕ 'Λάθος. Δώσε μη αρνητικό ποσό:'
        ΔΙΑΒΑΣΕ λογ[i, j]
      ΤΕΛΟΣ_ΕΠΑΝΑΛΗΨΗΣ
    ΤΕΛΟΣ_ΕΠΑΝΑΛΗΨΗΣ
  ΤΕΛΟΣ_ΕΠΑΝΑΛΗΨΗΣ

  ΓΡΑΨΕ 'α. Τραπέζια χωρίς άτομα:'
  βρΑ <- ΨΕΥΔΗΣ
  ΓΙΑ i ΑΠΟ 1 ΜΕΧΡΙ 16
    ΑΝ π[i] = 0 ΤΟΤΕ
      ΓΡΑΨΕ 'Τραπέζι ', i
      βρΑ <- ΑΛΗΘΗΣ
    ΤΕΛΟΣ_ΑΝ
  ΤΕΛΟΣ_ΕΠΑΝΑΛΗΨΗΣ
  ΑΝ βρΑ = ΨΕΥΔΗΣ ΤΟΤΕ
    ΓΡΑΨΕ 'Δεν υπάρχουν τραπέζια χωρίς άτομα.'
  ΤΕΛΟΣ_ΑΝ

  ΓΡΑΨΕ 'β. Συνολικός λογαριασμός τραπεζιού:'
  ΓΡΑΨΕ 'Δώσε αριθμό τραπεζιού (1-16):'
  ΔΙΑΒΑΣΕ αρΤραπ
  ΟΣΟ αρΤραπ < 1 Η αρΤραπ > 16 ΕΠΑΝΑΛΑΒΕ
    ΓΡΑΨΕ 'Λάθος αριθμός. Δώσε από 1 έως 16:'
    ΔΙΑΒΑΣΕ αρΤραπ
  ΤΕΛΟΣ_ΕΠΑΝΑΛΗΨΗΣ

  συνΛογ <- 0.0
  ΓΙΑ j ΑΠΟ 1 ΜΕΧΡΙ π[αρΤραπ]
    συνΛογ <- συνΛογ + λογ[αρΤραπ, j]
  ΤΕΛΟΣ_ΕΠΑΝΑΛΗΨΗΣ
  ΓΡΑΨΕ 'Ο συνολικός λογαριασμός του τραπεζιού ', αρΤραπ, ' είναι: ', συνΛογ

  ΓΡΑΨΕ 'γ. Τραπέζια με 5 άτομα και τουλάχιστον έναν μηδενικό λογαριασμό:'
  βρΓ <- ΨΕΥΔΗΣ
  ΓΙΑ i ΑΠΟ 1 ΜΕΧΡΙ 16
    ΑΝ π[i] = 5 ΤΟΤΕ
      έχειΜηδέν <- ΨΕΥΔΗΣ
      ΓΙΑ j ΑΠΟ 1 ΜΕΧΡΙ 5
        ΑΝ λογ[i, j] = 0 ΤΟΤΕ
          έχειΜηδέν <- ΑΛΗΘΗΣ
        ΤΕΛΟΣ_ΑΝ
      ΤΕΛΟΣ_ΕΠΑΝΑΛΗΨΗΣ
      ΑΝ έχειΜηδέν = ΑΛΗΘΗΣ ΤΟΤΕ
        ΓΡΑΨΕ 'Τραπέζι ', i
        βρΓ <- ΑΛΗΘΗΣ
      ΤΕΛΟΣ_ΑΝ
    ΤΕΛΟΣ_ΑΝ
  ΤΕΛΟΣ_ΕΠΑΝΑΛΗΨΗΣ

  ΑΝ βρΓ = ΨΕΥΔΗΣ ΤΟΤΕ
    ΓΡΑΨΕ 'Δεν βρέθηκε τέτοιο τραπέζι.'
  ΤΕΛΟΣ_ΑΝ
