ΛΥΣΗ: ΘΕΜΑ Γ (Πίνακες & Υποπρογράμματα)

Ενδεικτική Λύση: ΘΕΜΑ Γ

ΠΡΟΓΡΑΜΜΑ Θέμα_Γ
ΜΕΤΑΒΛΗΤΕΣ
  ΑΚΕΡΑΙΕΣ: ι, συχνότητα[100], ελάχιστηΣυχνότητα, ΘΕΣΗ
  ΧΑΡΑΚΤΗΡΕΣ: ΛΕΞΕΙΣ[100], λέξηΚειμένου
  ΛΟΓΙΚΕΣ: ΒΡΕΘΗΚΕ
ΑΡΧΗ
  ! Γ2. Ανάγνωση 100 ναυτικών λέξεων χωρίς διπλότυπα
  ΓΙΑ ι ΑΠΟ 1 ΜΕΧΡΙ 100
    ΑΡΧΗ_ΕΠΑΝΑΛΗΨΗΣ
      ΓΡΑΨΕ 'Δώσε ναυτική λέξη:'
      ΔΙΑΒΑΣΕ ΛΕΞΕΙΣ[ι]
      ! Αναζήτηση στις προηγούμενες καταχωρημένες λέξεις (1 έως ι-1)
      ΚΑΛΕΣΕ ΑΝΑΖΗΤΗΣΗ(ΛΕΞΕΙΣ, ι - 1, ΛΕΞΕΙΣ[ι], ΒΡΕΘΗΚΕ, ΘΕΣΗ)
    ΜΕΧΡΙΣ_ΟΤΟΥ ΒΡΕΘΗΚΕ = ΨΕΥΔΗΣ
    
    ! Αρχικοποίηση του πίνακα συχνοτήτων
    συχνότητα[ι] <- 0
  ΤΕΛΟΣ_ΕΠΑΝΑΛΗΨΗΣ

  ! Γ3. Ανάγνωση κειμένου και καταγραφή συχνοτήτων
  ΓΡΑΨΕ 'Δώσε λέξη κειμένου (ή ΤΕΛΟΣ_ΚΕΙΜΕΝΟΥ):'
  ΔΙΑΒΑΣΕ λέξηΚειμένου
  ΟΣΟ λέξηΚειμένου <> 'ΤΕΛΟΣ_ΚΕΙΜΕΝΟΥ' ΕΠΑΝΑΛΑΒΕ
    ΚΑΛΕΣΕ ΑΝΑΖΗΤΗΣΗ(ΛΕΞΕΙΣ, 100, λέξηΚειμένου, ΒΡΕΘΗΚΕ, ΘΕΣΗ)
    
    ΑΝ ΒΡΕΘΗΚΕ = ΑΛΗΘΗΣ ΤΟΤΕ
      συχνότητα[ΘΕΣΗ] <- συχνότητα[ΘΕΣΗ] + 1
    ΤΕΛΟΣ_ΑΝ
    
    ΓΡΑΨΕ 'Δώσε επόμενη λέξη:'
    ΔΙΑΒΑΣΕ λέξηΚειμένου
  ΤΕΛΟΣ_ΕΠΑΝΑΛΗΨΗΣ

  ! Γ4. Εύρεση σπανιότερων λέξεων (ελάχιστη συχνότητα εμφάνισης > 0)
  ελάχιστηΣυχνότητα <- -1
  ΓΙΑ ι ΑΠΟ 1 ΜΕΧΡΙ 100
    ΑΝ συχνότητα[ι] > 0 ΤΟΤΕ
      ΑΝ ελάχιστηΣυχνότητα = -1 ΤΟΤΕ
        ελάχιστηΣυχνότητα <- συχνότητα[ι]
      ΑΛΛΙΩΣ_ΑΝ συχνότητα[ι] < ελάχιστηΣυχνότητα ΤΟΤΕ
        ελάχιστηΣυχνότητα <- συχνότητα[ι]
      ΤΕΛΟΣ_ΑΝ
    ΤΕΛΟΣ_ΑΝ
  ΤΕΛΟΣ_ΕΠΑΝΑΛΗΨΗΣ

  ! Εμφάνιση αποτελεσμάτων
  ΑΝ ελάχιστηΣυχνότητα <> -1 ΤΟΤΕ
    ΓΡΑΨΕ 'Οι σπανιότερες ναυτικές λέξεις στο κείμενο (εμφανίστηκαν ', ελάχιστηΣυχνότητα, ' φορές) είναι:'
    ΓΙΑ ι ΑΠΟ 1 ΜΕΧΡΙ 100
      ΑΝ συχνότητα[ι] = ελάχιστηΣυχνότητα ΤΟΤΕ
        ΓΡΑΨΕ ΛΕΞΕΙΣ[ι]
      ΤΕΛΟΣ_ΑΝ
    ΤΕΛΟΣ_ΕΠΑΝΑΛΗΨΗΣ
  ΑΛΛΙΩΣ
    ΓΡΑΨΕ 'Δεν βρέθηκε καμία ναυτική λέξη στο κείμενο.'
  ΤΕΛΟΣ_ΑΝ
ΤΕΛΟΣ_ΠΡΟΓΡΑΜΜΑΤΟΣ


! Γ1. Κατασκευή υποπρογράμματος ΑΝΑΖΗΤΗΣΗ
ΔΙΑΔΙΚΑΣΙΑ ΑΝΑΖΗΤΗΣΗ(Π, Ν, Χ, ΒΡΕΘΗΚΕ, ΘΕΣΗ)
ΜΕΤΑΒΛΗΤΕΣ
  ΧΑΡΑΚΤΗΡΕΣ: Π[100], Χ
  ΑΚΕΡΑΙΕΣ: Ν, ΘΕΣΗ, ι
  ΛΟΓΙΚΕΣ: ΒΡΕΘΗΚΕ
ΑΡΧΗ
  ΒΡΕΘΗΚΕ <- ΨΕΥΔΗΣ
  ΘΕΣΗ <- 0
  ι <- 1
  ΟΣΟ ι <= Ν ΚΑΙ ΒΡΕΘΗΚΕ = ΨΕΥΔΗΣ ΕΠΑΝΑΛΑΒΕ
    ΑΝ Π[ι] = Χ ΤΟΤΕ
      ΒΡΕΘΗΚΕ <- ΑΛΗΘΗΣ
      ΘΕΣΗ <- ι
    ΑΛΛΙΩΣ
      ι <- ι + 1
    ΤΕΛΟΣ_ΑΝ
  ΤΕΛΟΣ_ΕΠΑΝΑΛΗΨΗΣ
ΤΕΛΟΣ_ΔΙΑΔΙΚΑΣΙΑΣ
