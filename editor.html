ΠΡΟΕΠΙΣΚΟΠΗΣΗ

ΕΝΤΟΠΙΣΜΟΣ ΛΑΘΩΝ (ΜΑΘΗΤΕΣ ΚΑΙ ΒΑΘΜΟΛΟΓΙΕΣ)

Δίνεται το παρακάτω πρόγραμμα το οποίο διαβάζει τα ονόματα των μαθητών και τους βαθμούς τους στα Μαθηματικά, στις ΑΟΘ και στη Γλώσσα. Υπολογίζει τον μεγαλύτερο βαθμό κάθε μαθητή, τον μέσο όρο του και στο τέλος τον γενικό μέσο όρο της τάξης.

Ωστόσο, το πρόγραμμα περιέχει συντακτικά λάθη, λογικά λάθη, καθώς και λάθη κατά την εκτέλεση.

1.  ΠΡΟΓΡΑΜΜΑ last2026bug
2.  ΜΕΤΑΒΛΗΤΕΣ
3.    ΑΚΕΡΑΙΕΣ: πληθοςΜαθητών, σ, συνολοΤάξης
4.    ΠΡΑΓΜΑΤΙΚΕΣ: μαθ, αοθ, γλωσ, μαξ, μεσοςΟροςΤαξης
5.    ΧΑΡΑΚΤΗΡΕΣ: ον
6.  ΑΡΧΗ
7.    πληθοςΜαθητών <- 1
8.    ΑΡΧΗ_ΕΠΑΝΑΛΗΨΗΣ
9.      ΓΡΑΨΕ 'Δώστε όνομα ή τέλος'
10.     ΔΙΑΒΑΣΕ ον
11.     ΓΡΑΨΕ 'Δώστε στα μαθηματικά, αοθ, γλώσσα'
12.     ΔΙΑΒΑΣΕ μαθ, αοθ, γλωσ
13.     μαξ <- μαθ
14.     μαξΜαθ <- 'μαθηματικά'
15.     ΑΝ αοθ > μαξ ΤΟΤΕ
16.       μαξ <- αοθ
17.       μαξΜαθ <- 'αοθ'
18.     ΑΛΛΙΩΣ γλωσ > μαξ ΤΟΤΕ
19.       μαξ <- αοθ
20.       μαξΜαθ <- 'αοθ'
21.     ΤΕΛΟΣ_ΑΝ
22.     ΓΡΑΨΕ 'Ο μεγαλύτερος βαθμός ήταν στο ', μαξΜαθ
23.     μο <- μαθ + αοθ + γλωσ/ 3
24.     ΓΡΑΨΕ 'Ο μέσος όρος του ', ον, ' είναι ', μο
25.     πληθοςΜαθητών <- πληθοςΜαθητών + μο
26.     συνολοΤάξης <- συνολοΤάξης + 1
27.   ΜΕΧΡΙΣ_ΟΤΟΥ ον = 'τέλος'
28.   μεσοςΟροςΤαξης <- συνολοΤάξης/ πληθοςΜαθητών
29.   ΓΡΑΨΕ 'Ο μέσος όρος της τάξης είναι ', μεσοςΟροςΤαξης
30. ΤΕΛΟΣ_ΠΡΟΓΡΑΜΜΑΤΟΣ

Να εντοπίσετε τα λάθη αναφέροντας τον αριθμό της γραμμής, να τα διορθώσετε και να τα κατατάξετε σε μία από τις παρακάτω κατηγορίες:

  • Συντακτικό λάθος
  • Λογικό λάθος
  • Λάθος κατά την εκτέλεση

Στη συνέχεια, να γράψετε ολόκληρο το πρόγραμμα στη σωστή του μορφή.
