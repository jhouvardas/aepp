    ΔΙΑΒΑΣΕ x
    Σ <- Σ + ...(3)...
ΤΕΛΟΣ_ΕΠΑΝΑΛΗΨΗΣ

Άσκηση 2 (Μέσος Όρος με συνθήκη τερματισμού)

Να συμπληρώσετε τα κενά ώστε να υπολογίζεται ο μέσος όρος των θετικών αριθμών που δίνονται, μέχρι να δοθεί το μηδέν (0).

Π <- ...(1)...
ΔΙΑΒΑΣΕ x
ΟΣΟ x ...(2)... 0 ΕΠΑΝΑΛΑΒΕ
    ΑΝ x > 0 ΤΟΤΕ
        Σ <- Σ + x
        Π <- Π + ...(3)...
    ΤΕΛΟΣ_ΑΝ
    ΔΙΑΒΑΣΕ x
ΤΕΛΟΣ_ΕΠΑΝΑΛΗΨΗΣ
ΑΝ Π > 0 ΤΟΤΕ
    ΜΟ <- ...(4)... / Π
    ΓΡΑΨΕ ΜΟ
ΤΕΛΟΣ_ΑΝ
